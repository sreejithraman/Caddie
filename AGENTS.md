# Repository guidance

## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues; external pull requests are not currently a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

The standard five-role triage vocabulary is used unchanged. See `docs/agents/triage-labels.md`.

### Domain docs

Caddie uses a single-context domain layout. See `docs/agents/domain.md`.
